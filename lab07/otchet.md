## Используемые методы и алгоритмы

### 1. Собственный ГПСЧ (класс `MultiKongGen`)

Линейный конгруэнтный генератор с параметрами:
\[
x_{n+1} = (\beta \cdot x_n) \mod M, \quad \beta = 2^{32}+3,\; M = 2^{63}.
\]
Начальное состояние (seed) привязывается к текущему времени в наносекундах.  
Метод `anabios()` возвращает число \( u \in [0,1) \).

### 2. Моделирование дискретной случайной величины

**Метод обратной функции (кумулятивных вероятностей):**  
- Строим кумулятивный ряд \( F(x_i) = \sum_{j=1}^i p_j \).  
- Генерируем \( u \sim U(0,1) \).  
- Находим наименьшее \( x_i \), для которого \( u < F(x_i) \).  

Пользователь задаёт любые неотрицательные веса в интерфейсе; они нормируются так, что сумма вероятностей становится равной 1.

### 3. Моделирование нормальной случайной величины \( N(0,1) \)

**Метод суммирования 12 независимых равномерных величин (ЦПТ):**  
- Генерируем 12 чисел \( u_1, \dots, u_{12} \) из \( U(0,1) \).  
- \( S = \sum_{i=1}^{12} u_i \).  
- В силу центральной предельной теоремы \( S \approx N(6,1) \).  
- Вычитая 6, получаем \( X = S - 6 \sim N(0,1) \).  

Для получения \( N(\mu, \sigma^2) \) используется преобразование \( Y = \mu + \sigma X \).

### 4. Критерий \(\chi^2\) Пирсона

- **Для ДСВ:** значения разбиты по индивидуальным дискретным исходам.  
  Статистика  
  \[
  \chi^2 = \sum_{i=1}^{k} \frac{(f_i^{\text{набл}} - f_i^{\text{ож}})^2}{f_i^{\text{ож}}},
  \quad f_i^{\text{ож}} = N \cdot p_i
  \]
  где \(k\) – количество значений.  
  Степени свободы \( df = k-1 \).  

- **Для НСВ:** непрерывная область разбивается на 6 интервалов (от \(-\infty\) до \(-2\), \(-2\) до \(-1\), \(-1\) до \(0\), \(0\) до \(1\), \(1\) до \(2\), \(2\) до \(+\infty\)).  
  Теоретические вероятности попадания в интервалы вычисляются через функцию стандартного нормального распределения \( \Phi(x) = \frac12\left(1 + \operatorname{erf}(x/\sqrt{2})\right) \).  
  Степени свободы \( df = 6-1 = 5 \).  

Уровень значимости \( \alpha = 0.05 \) используется во всех тестах.  
Критические значения взяты из стандартных таблиц:
- для \(df=4\) (ДСВ): \( \chi^2_{\text{крит}} = 9.488 \);
- для \(df=5\) (НСВ): \( \chi^2_{\text{крит}} = 11.07 \).

## Приложение: Листинг основных функций

```python
# rng.py – собственный ГПСЧ
class MultiKongGen:
    def __init__(self, seed=None):
        self.M = 2 ** 63
        self.beta = 2 ** 32 + 3
        if seed is None:
            seed = time.time_ns() % self.M
        self.x0 = seed
    def anabios(self):
        self.x0 = (self.beta * self.x0) % self.M
        return self.x0 / self.M
```
# logic.py – генерация и статистика
```python
def generate_discr_samples(rng, values, probs, n):
    cum = [sum(probs[:i+1]) for i in range(len(probs))]
    samples = []
    for _ in range(n):
        u = rng.anabios()
        for i, cp in enumerate(cum):
            if u < cp:
                samples.append(values[i])
                break
    return samples
```
