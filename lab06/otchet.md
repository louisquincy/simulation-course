## Часть 2. Нормальная случайная величина $N(\mu, \sigma^2)$

### 2.1. Параметры и метод генерации

В модифицированной программе добавлена возможность задавать параметры нормального распределения. Для эксперимента выбрана модель $X \sim N(\mu = 7, \sigma^2 = 3)$.  

Алгоритм генерации:
1. Согласно Центральной предельной теореме генерируется базовая стандартная величина $Z \sim N(0,1)$ методом суммирования 12 равномерно распределенных величин $U(0,1)$ с последующим вычитанием 6 (математического ожидания суммы).
2. Выполняется линейное преобразование для получения заданных параметров:  
   $$X = \mu + \sigma \cdot Z = 7 + \sqrt{3} \cdot Z$$

*Примечание:* В таблице результатов для нормальной СВ вычисляется **абсолютная погрешность** математического ожидания ($|M_{эмп} - \mu|$), а не относительная. Это сделано для избежания ошибки деления на ноль в случаях, когда пользователь задает $\mu = 0$ (стандартное нормальное распределение).

## Приложение: Листинг алгоритма генерации нормальной СВ с параметрами $\mu$ и $\sigma^2$

```python
def generate_norm_samples(rng: MultiKongGen, n: int, mean: float = 0.0, variance: float = 1.0):
    samples = []
    std_dev = math.sqrt(variance)
    for _ in range(n):
        # 1. Генерируем сумму 12 равномерных СВ U(0,1)
        u_sum = sum(rng.anabios() for _ in range(12))
        
        # 2. Центрируем (вычитаем мат. ожидание суммы) -> получаем Z ~ N(0, 1)
        z = u_sum - 6.0
        
        # 3. Линейное преобразование для получения X ~ N(mean, variance)
        x = mean + std_dev * z
        samples.append(x)
        
    return samples
```
