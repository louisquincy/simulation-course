### 1. Теоретическая часть

#### Мультипликативный конгруэнтный генератор

Мультипликативный конгруэнтный генератор (МКГ) — это детерминированный алгоритм генерации псевдослучайных чисел, являющийся частным случаем линейного конгруэнтного метода при нулевом приращении.

Рекуррентное соотношение МКГ:

$$x_i^* = (\beta \cdot x_{i-1}^*) \mod M$$

Нормировка для получения значений на отрезке $[0, 1)$:

$$x_i = \frac{x_i^*}{M}, \quad i = 1, 2, \ldots$$

Параметры генератора, использованные в работе:

| Параметр | Значение | Описание |
|----------|----------|----------|
| $M$ | $2^{63}$ | Модуль |
| $\beta$ | $2^{32} + 3$ | Множитель |
| $x_0^*$ | $x_0^* = \beta$ | Начальное значение |

---

### 2. Реализация

```python

def anabios(self):
    self.state = (self.beta * self.state) % self.M
    return self.state / self.M


def skill(generator_func, n=N):
    return [generator_func() for _ in range(n)]
```
