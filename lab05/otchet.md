## Используемые методы и алгоритмы

### 1. Собственный ГПСЧ (класс `MultiKongGen`)

Линейный конгруэнтный генератор:  
`x_{n+1} = (β * x_n) mod M`,  
β = 2³² + 3, M = 2⁶³.  
Начальное состояние (seed) инициализируется текущим временем (наносекунды), что обеспечивает разные последовательности при каждом запуске.  
Метод `anabios()` возвращает равномерно распределённое число в интервале `[0, 1)`.

### 2. Генератор Yes/No

Вероятность ответа «Yes» задаётся параметром `YES_PROB` (в работе принята равной 0.5, но может быть изменена пользователем).  
Алгоритм:  
- Генерируется `u = anabios()`.  
- Если `u < YES_PROB`, результат «Yes», иначе «No».

## Приложение: Листинг ключевых функций (из `logic.py`)

``` python
def yes_no_once(rng, yes_prob=0.5):
    return "Yes" if rng.anabios() < yes_prob else "No"
```
``` python
def gacha_pull_once(rng, names, weights):
    total = sum(weights)
    point = rng.anabios() * total
    cum = 0.0
    for name, w in zip(names, weights):
        cum += w
        if point < cum:
            return name
    return names[-1]
```
