# Отчет по лабораторной работе
## Тема: Клеточные автоматы. Моделирование лесных пожаров

---

### 1. Клеточные автоматы. Лесные пожары

**Задание:**  
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

**Требования:**
- реализовать **не менее трёх дополнительных правил** поведения системы.

---

### 2. Инструментарий
Для реализации выбран язык **Python** и следующий стек технологий:
*   **PyQt5**: Для создания современного оконного интерфейса и отрисовки поля.
*   **NumPy**: Для хранения состояний автомата в виде матриц (оптимизация памяти).
*   **Numba (`@njit`)**: JIT-компилятор. Используется для ускорения вычислений логики автомата (достигнута высокая производительность на сетке 200x200).

---

### 3. Описание модели (Логика)
Используется **двумерный клеточный автомат** с окрестностью Мура (8 соседей).
**Состояния клетки:**
`Земля` (0), `Лес` (1), `Огонь` (2), `Угли/Пепел` (3), `Вода` (4), `Скалы` (5).

#### Реализованные дополнительные правила (Уникальность):

1.  **Векторный ветер (Wind Dynamics):**
    *   Влияние ветра рассчитывается через **скалярное произведение** вектора направления ветра и вектора распространения огня.
    *   Учитывается угол (0-360°) и сила ветра. Попутный ветер значительно увеличивает вероятность возгорания соседа.

2.  **Рельеф местности (Elevation Logic):**
    *   Генерируется карта высот (шум + сглаживание).
    *   Реализована физика конвекции: огонь распространяется **вверх по склону** быстрее, чем вниз.
    *   Введены «Скалы» (высокогорье), которые являются естественной преградой для огня.

3.  **Термодинамика почвы (Ash Cooling):**
    *   После сгорания клетка переходит в состояние «Угли» (Ash).
    *   Введен таймер остывания: на горячей земле новый лес не растет, пока таймер не обнулится.

---

### 4. Графический интерфейс (GUI)
Разработано приложение на базе `QMainWindow`.
**Функционал:**
*   **Canvas:** Отрисовка поля в реальном времени с цветовой кодировкой (RGB).
*   **Панель управления:**
    *   Кнопки: Старт/Пауза, Генерация новой карты.
    *   Слайдеры: Скорость роста леса, Вероятность возгорания.
    *   Настройки ветра: Ползунки направления и силы.
    *   Легенда: Цветовое обозначение типов местности.

---

### 5. Результаты
В ходе работы создана стабильная имитационная модель.
*   Достигнута высокая производительность (30+ FPS) благодаря использованию Numba.
*   Визуализация корректно отображает ландшафт (вода в низинах, скалы на вершинах).
*   Логика поведения огня соответствует физическим ожиданиям (огибание препятствий, ускорение по ветру).
